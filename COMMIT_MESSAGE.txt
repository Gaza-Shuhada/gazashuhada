## ğŸ› Critical Bug Fix: Form Submission Hanging

Fixed a critical bug where all form submissions would hang, leaving buttons 
disabled and preventing users from seeing success/error messages.

### Root Cause
- Using `response.json()` directly would hang on empty/invalid responses
- Synchronous `fetchX()` calls blocked UI updates
- Success handlers ran async operations before scrolling/showing messages

### Solution Applied System-Wide
1. **Safe JSON Parsing**: Replace all `response.json()` with:
   ```typescript
   const text = await response.text();
   const data = text ? JSON.parse(text) : {};
   ```

2. **Non-blocking Refreshes**: Changed from:
   ```typescript
   fetchSubmissions(); // Blocks
   ```
   To:
   ```typescript
   fetchSubmissions().catch(err => console.error(...)); // Non-blocking
   ```

3. **Immediate UI Feedback**: Move `window.scrollTo()` to run FIRST before 
   any async operations
